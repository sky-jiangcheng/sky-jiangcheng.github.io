## 后端开发

- **《Effective Java》** - Joshua Bloch，Java 开发必读经典
- **《Java 并发编程实战》** - Brian Goetz，深入理解 Java 并发
- **《Spring 实战》** - Craig Walls，Spring 框架权威指南
- **《高性能 MySQL》** - Baron Schwartz，MySQL 性能优化宝典

## 架构与设计

- **《企业应用架构模式》** - Martin Fowler，企业架构模式经典
- **《领域驱动设计》** - Eric Evans，DDD 奠基之作
- **《设计模式：可复用面向对象软件的基础》** - GoF，设计模式圣经
- **《微服务设计》** - Sam Newman，微服务架构实践指南

## 团队与工程

- **《人月神话》** - Frederick Brooks，软件工程管理经典
- **《重构：改善既有代码的设计》** - Martin Fowler，代码重构指南
- **《代码整洁之道》** - Robert C. Martin，编写高质量代码
- **《持续交付》** - Jez Humble，DevOps 实践手册
