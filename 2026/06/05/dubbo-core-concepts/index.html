Dubbo 分布式服务框架核心概念与实践

2026-06-05 Dubbo Dubbo, Java, RPC, 微服务 评论

前言

Dubbo 是阿里巴巴开源的高性能 Java RPC 框架，在金融和大型企业级系统中被广泛使用。它提供了服务注册发现、负载均衡、容错和监控等完整的服务治理能力。

1. Dubbo 核心架构

Dubbo 的架构包含五个核心角色：

  • Provider：暴露服务的服务提供方
  • Consumer：调用远程服务的服务消费方
  • Registry：服务注册与发现中心
  • Monitor：统计服务的调用次数和调用时间
  • Container：服务运行容器

策略 说明

roundrobin 轮询，按公约权重轮询
leastactive 最少活跃调用数
